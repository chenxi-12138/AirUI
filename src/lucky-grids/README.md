# lucky-grids 跑马灯转盘 vue版本

### 介绍

 lucky-grids是基于`lucky-canvas/vue`依赖进行二次封装的组件

### 示例

### 引入

```js
import Vue from 'vue';
import {  lucky-grids } from 'AirUI';

Vue.use( lucky-grids);
```
